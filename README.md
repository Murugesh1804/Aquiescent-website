Installation

Prerequisites

Ensure you have the following installed:

  Node.js (>=16.x)
  
  npm or yarn

Steps to Run Locally

1. Clone the repository:

  git clone <repo-link>
  cd Acquiescent-website

2. Install dependencies:

  npm install  
 

3. Start the development server:

  npm run dev


Open http://localhost:3000 in your browser.
